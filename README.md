A web-based Attendance Management System built with Python and Flask, designed to simplify student attendance tracking. The application allows users to mark students as present or absent, add new students with their name, roll number, and branch, and view attendance statistics filtered by date and branch. It also generates detailed attendance reports showing percentage and status for each student. Built with a clean web interface and easy local setup, the system runs entirely in the browser via a Flask development server. Ideal for academic institutions looking for a lightweight, dependency-minimal solution to manage and monitor student attendance efficiently.
